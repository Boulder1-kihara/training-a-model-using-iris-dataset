# Iris Flower Classification Model

This project builds a machine learning model to classify Iris flowers into three species (setosa, versicolor, virginica) based on their measurements.

## Project Overview
- **Dataset:** Iris Dataset (Scikit-Learn)
- **Model:** Decision Tree Classifier (or whichever model you used)
- **Tools:** Python, Pandas, Scikit-Learn
